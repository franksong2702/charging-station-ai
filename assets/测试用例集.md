## 二、故障处理场景测试（8条）

| 项目 | 内容 |

| **预期结果** | 应提供三种停止方法：车内停止、充电桩停止、急停按钮 |

### TC-008: 充电停不下来（特斯拉）
| 项目 | 内容 |
|------|------|
| **测试问题** | 特斯拉充电停不下来怎么办，App点了停止还是继续 |
| **预期场景** | 故障处理 |
| **预期结果** | 应针对特斯拉提供解决方案 |

### TC-009: 充电枪拔不出来（基本）
| 项目 | 内容 |
|------|------|
| **测试问题** | 充电结束了，但是充电枪卡住拔不出来 |
| **预期场景** | 故障处理 |
| **预期结果** | 应提供解锁车辆、正确姿势拔枪等方法 |

### TC-010: 充电枪拔不出来（比亚迪）
| 项目 | 内容 |
|------|------|
| **测试问题** | 比亚迪的车，充电枪拔不出来，怎么办啊？ |
| **预期场景** | 故障处理 |
| **预期结果** | 应针对比亚迪提供解决方案（App先结束充电） |

### TC-011: 无法启动充电
| 项目 | 内容 |
|------|------|
| **测试问题** | 扫码了但是充不了电，怎么回事？ |
| **预期场景** | 故障处理 |
| **预期结果** | 应提供检查清单：充电枪是否插好、车辆是否就绪、支付是否正常 |

### TC-012: 错误代码查询
| 项目 | 内容 |
|------|------|
| **测试问题** | 充电桩显示E01错误代码，什么意思？ |
| **预期场景** | 故障处理 |
| **预期结果** | 应解释E01=充电枪未连接，建议重新插拔 |

### TC-013: 充电速度慢
| 项目 | 内容 |
|------|------|
| **测试问题** | 充电速度好慢啊，以前不是这样的 |
| **预期场景** | 故障处理 |
| **预期结果** | 应解释可能原因：电量过高、电池温度低、充电桩功率限制 |

### TC-014: 复杂故障描述
| 项目 | 内容 |
|------|------|
| **测试问题** | 充电到一半突然停了，显示屏显示故障，充电枪也拔不出来，怎么办？ |
| **预期场景** | 故障处理 |
| **预期结果** | 应提供综合解决方案或建议联系客服 |
