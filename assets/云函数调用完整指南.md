## 二、API 接口详细规范

### 1. 接口地址

```
POST http://your-server-ip:5000/run
```

**⚠️ 重要提示**：
- 如果部署在本地：需要公网IP或内网穿透
- 如果部署在云服务器：使用服务器公网IP
- 端口默认：5000（可在启动时修改）

### 2. 请求格式

#### 请求头（Headers）
```
Content-Type: application/json
```

#### 请求体（Body）
```json
{
  "user_message": "用户发送的消息内容"
}
```

**参数说明**：

| 参数名 | 类型 | 必填 | 说明 | 示例值 |
|--------|------|------|------|--------|
| `user_message` | string | ✅ 是 | 用户发送的消息 | "充电停不下来了怎么办？" |

### 3. 响应格式

#### 成功响应（200）
```json
{
  "reply_content": "您好！如果充电停不下来，请尝试以下方法...",
  "run_id": "383b7167-7f34-4346-8020-ee274fd3d710"
}
```

**字段说明**：

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `reply_content` | string | AI客服的回复内容 |
| `run_id` | string | 本次运行的唯一ID（用于追踪和取消） |

#### 错误响应

**400 错误（参数错误）**
```json
{
  "detail": "Invalid JSON format"
}
```

**500 错误（服务器错误）**
```json
{
  "detail": {
    "error_code": "INTERNAL_ERROR",
    "error_message": "服务器内部错误",
    "stack_trace": "..."
  }
}
```

**504 错误（超时）**
```json
{
  "status": "timeout",
  "run_id": "xxx",
  "message": "Execution timeout: exceeded 900 seconds"
}
```

## 三、腾讯云函数完整代码

### 方式1：基础版（无认证）

```python
# -*- coding: utf-8 -*-
"""
腾讯云函数 - 充电桩智能客服调用
版本：基础版（无认证）
"""

import json
import requests
import logging
from typing import Dict, Any

logger = logging.getLogger()
logger.setLevel(logging.INFO)

# ==================== 配置区 ====================
# TODO: 替换为您的服务器公网地址
COZE_API_URL = "http://your-server-ip:5000/run"
# 示例：COZE_API_URL = "http://123.45.67.89:5000/run"
# ==================== 配置区结束 ====================


def main_handler(event: Dict[str, Any], context: Any) -> Dict[str, Any]:
    """
    云函数入口
    """
    try:
        # 1. 解析请求
        if 'body' in event:
            body = event.get('body', '{}')
            if isinstance(body, str):
                request_data = json.loads(body)
            else:
                request_data = body
        else:
            request_data = event
        
        # 2. 提取用户消息
        user_message = request_data.get('user_message') or \
                       request_data.get('message') or \
                       request_data.get('content')
        
        if not user_message:
            return {
                "statusCode": 400,
                "headers": {
                    "Content-Type": "application/json",
                    "Access-Control-Allow-Origin": "*"
                },
                "body": json.dumps({
                    "success": False,
                    "error": "缺少必填参数：user_message"
                }, ensure_ascii=False)
            }
        
        logger.info(f"用户消息: {user_message}")
        
        # 3. 调用AI工作流API
        response = requests.post(
            COZE_API_URL,
            json={"user_message": user_message},
            headers={"Content-Type": "application/json"},
            timeout=30
        )
        
        # 4. 处理响应
        if response.status_code == 200:
            result = response.json()
            return {
                "statusCode": 200,
                "headers": {
                    "Content-Type": "application/json",
                    "Access-Control-Allow-Origin": "*"
                },
                "body": json.dumps({
                    "success": True,
                    "reply_content": result.get('reply_content', ''),
                    "run_id": result.get('run_id', '')
                }, ensure_ascii=False)
            }
        else:
            logger.error(f"API调用失败: {response.status_code} - {response.text}")
            return {
                "statusCode": 500,
                "headers": {
                    "Content-Type": "application/json",
                    "Access-Control-Allow-Origin": "*"
                },
                "body": json.dumps({
                    "success": False,
                    "error": f"API调用失败: {response.status_code}",
                    "detail": response.text
                }, ensure_ascii=False)
            }
            
    except Exception as e:
        logger.error(f"异常: {e}", exc_info=True)
        return {
            "statusCode": 500,
            "headers": {
                "Content-Type": "application/json",
                "Access-Control-Allow-Origin": "*"
            },
            "body": json.dumps({
                "success": False,
                "error": str(e)
            }, ensure_ascii=False)
        }
```

### 方式2：企业微信对接版（完整版）

```python
# -*- coding: utf-8 -*-
"""
腾讯云函数 - 充电桩智能客服 + 企业微信对接
版本：完整版（含企业微信推送）
"""

import json
import requests
import logging
from typing import Dict, Any, Optional

logger = logging.getLogger()
logger.setLevel(logging.INFO)

# ==================== 配置区 ====================
# AI工作流API地址
COZE_API_URL = "http://your-server-ip:5000/run"

# 企业微信机器人Webhook地址（可选）
WECHAT_WEBHOOK_URL = ""  # 留空则不推送企业微信

# 是否启用认证（可选）
ENABLE_AUTH = False
AUTH_TOKEN = "your_secret_token_here"
# ==================== 配置区结束 ====================


def send_wechat_message(webhook_url: str, content: str) -> bool:
    """
    发送消息到企业微信
    
    Args:
        webhook_url: 企业微信机器人Webhook地址
        content: 消息内容
    
    Returns:
        是否发送成功
    """
    if not webhook_url:
        return False
    
    try:
        payload = {
            "msgtype": "text",
            "text": {
                "content": content
            }
        }
        response = requests.post(webhook_url, json=payload, timeout=5)
        return response.status_code == 200
    except Exception as e:
        logger.error(f"企业微信发送失败: {e}")
        return False


def call_ai_workflow(user_message: str) -> Dict[str, Any]:
    """
    调用AI工作流
    
    Args:
        user_message: 用户消息
    
    Returns:
        API响应结果
    """
    response = requests.post(
        COZE_API_URL,
        json={"user_message": user_message},
        headers={"Content-Type": "application/json"},
        timeout=30
    )
    
    if response.status_code == 200:
        return {
            "success": True,
            "data": response.json()
        }
    else:
        return {
            "success": False,
            "error": f"API调用失败: {response.status_code}",
            "detail": response.text
        }


def main_handler(event: Dict[str, Any], context: Any) -> Dict[str, Any]:
    """
    云函数入口
    支持两种模式：
    1. 直接调用：传入 user_message
    2. 企业微信回调：接收企业微信推送的消息
    """
    try:
        # 解析请求
        if 'body' in event:
            body = event.get('body', '{}')
            if isinstance(body, str):
                request_data = json.loads(body)
            else:
                request_data = body
        else:
            request_data = event
        
        # ==================== 认证检查（可选）====================
        if ENABLE_AUTH:
            # 从header或body中获取token
            headers = event.get('headers', {})
            token = headers.get('Authorization', '').replace('Bearer ', '')
            token = token or request_data.get('token')
            
            if token != AUTH_TOKEN:
                return {
                    "statusCode": 401,
                    "headers": {"Content-Type": "application/json"},
                    "body": json.dumps({
                        "success": False,
                        "error": "认证失败"
                    }, ensure_ascii=False)
                }
        # ==================== 认证检查结束 ====================
        
        # 提取用户消息
        # 支持多种字段名
        user_message = (
            request_data.get('user_message') or
            request_data.get('message') or
            request_data.get('content') or
            request_data.get('Content')  # 企业微信格式
        )
        
        if not user_message:
            return {
                "statusCode": 400,
                "headers": {
                    "Content-Type": "application/json",
                    "Access-Control-Allow-Origin": "*"
                },
                "body": json.dumps({
                    "success": False,
                    "error": "缺少必填参数：user_message"
                }, ensure_ascii=False)
            }
        
        logger.info(f"收到用户消息: {user_message}")
        
        # 调用AI工作流
        result = call_ai_workflow(user_message)
        
        if not result['success']:
            return {
                "statusCode": 500,
                "headers": {
                    "Content-Type": "application/json",
                    "Access-Control-Allow-Origin": "*"
                },
                "body": json.dumps(result, ensure_ascii=False)
            }
        
        reply_content = result['data'].get('reply_content', '')
        run_id = result['data'].get('run_id', '')
        
        logger.info(f"AI回复: {reply_content}")
        
        # 推送到企业微信（如果配置了）
        if WECHAT_WEBHOOK_URL:
            wechat_success = send_wechat_message(WECHAT_WEBHOOK_URL, reply_content)
            logger.info(f"企业微信推送: {'成功' if wechat_success else '失败'}")
        
        # 返回响应
        return {
            "statusCode": 200,
            "headers": {
                "Content-Type": "application/json",
                "Access-Control-Allow-Origin": "*",
                "Access-Control-Allow-Methods": "POST, OPTIONS",
                "Access-Control-Allow-Headers": "Content-Type, Authorization"
            },
            "body": json.dumps({
                "success": True,
                "reply_content": reply_content,
                "run_id": run_id,
                "wechat_sent": bool(WECHAT_WEBHOOK_URL)
            }, ensure_ascii=False)
        }
        
    except requests.Timeout:
        logger.error("API调用超时")
        return {
            "statusCode": 504,
            "headers": {"Content-Type": "application/json"},
            "body": json.dumps({
                "success": False,
                "error": "API调用超时"
            }, ensure_ascii=False)
        }
    
    except json.JSONDecodeError as e:
        logger.error(f"JSON解析错误: {e}")
        return {
            "statusCode": 400,
            "headers": {"Content-Type": "application/json"},
            "body": json.dumps({
                "success": False,
                "error": "JSON格式错误"
            }, ensure_ascii=False)
        }
    
    except Exception as e:
        logger.error(f"未知异常: {e}", exc_info=True)
        return {
            "statusCode": 500,
            "headers": {"Content-Type": "application/json"},
            "body": json.dumps({
                "success": False,
                "error": f"服务器错误: {str(e)}"
            }, ensure_ascii=False)
        }
```

## 六、安全建议

### ⚠️ 当前状态：无认证

**风险**：任何人知道地址都能调用您的API

### 解决方案

#### 方案1：添加Token认证（推荐）

修改云函数代码：

```python
# 在配置区添加
ENABLE_AUTH = True
AUTH_TOKEN = "your_random_secret_token_here"

# 在 main_handler 中添加认证检查
if ENABLE_AUTH:
    headers = event.get('headers', {})
    token = headers.get('Authorization', '').replace('Bearer ', '')
    
    if token != AUTH_TOKEN:
        return {
            "statusCode": 401,
            "body": json.dumps({"error": "认证失败"})
        }
```

调用时添加Token：
```python
headers = {"Authorization": "Bearer your_random_secret_token_here"}
```

#### 方案2：IP白名单

在API网关配置中设置IP白名单，只允许企业微信服务器IP访问。

#### 方案3：企业微信签名验证

验证请求是否来自企业微信服务器。

## 八、常见问题

### Q1: 如何获取服务器公网IP？

**A**: 购买云服务器后，在控制台查看公网IP地址。

### Q2: 端口5000无法访问？

**A**: 需要在云服务器安全组中开放5000端口：
1. 登录腾讯云控制台
2. 找到云服务器 → 安全组
3. 添加入站规则：允许TCP 5000端口

### Q3: 云函数超时怎么办？

**A**: 
1. 增加云函数超时时间（最大900秒）
2. 优化AI工作流响应速度
3. 使用流式接口 `/stream_run`

### Q4: 如何查看日志？

**A**: 
- 云函数日志：腾讯云函数控制台 → 日志查询
- 工作流日志：服务器上的 `/app/work/logs/bypass/app.log`
