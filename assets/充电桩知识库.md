# 充电桩智能客服知识库

## 一、扫码使用指南

### 1.1 首次使用须知

**用户常见问题**：看到很多二维码，不知道扫哪个码？

**解答流程**：
1. 首先询问用户是什么品牌的车
2. 根据车型指导扫码位置

### 1.2 不同品牌车型扫码指引

#### 特斯拉（Tesla）
- **扫码位置**：充电桩正面显示屏下方
- **二维码特征**：黑白二维码，旁边有Tesla标识
- **操作步骤**：
  1. 找到充电桩正面显示屏
  2. 在显示屏下方找到二维码
  3. 使用微信或小程序扫码
  4. 按提示完成充电

#### 比亚迪（BYD）
- **扫码位置**：充电桩侧面或顶部
- **二维码特征**：彩色二维码，有BYD标识
- **操作步骤**：
  1. 查看充电桩侧面或顶部
  2. 找到彩色二维码
  3. 扫码后选择充电时长
  4. 确认开始充电

#### 蔚来（NIO）
- **扫码位置**：充电桩正面中央位置
- **二维码特征**：大型二维码，蓝色背景
- **操作步骤**：
  1. 查看充电桩正面中央
  2. 扫描蓝色背景二维码
  3. 使用蔚来App或小程序
  4. 按提示操作

#### 小鹏（XPeng）
- **扫码位置**：充电桩顶部显示屏旁
- **二维码特征**：小型二维码，绿色边框
- **操作步骤**：
  1. 抬头查看充电桩顶部
  2. 找到绿色边框二维码
  3. 扫码后自动跳转小程序
  4. 选择充电模式开始

#### 理想（Li Auto）
- **扫码位置**：充电桩正面右侧
- **二维码特征**：圆形二维码，绿色背景
- **操作步骤**：
  1. 查看充电桩正面右侧
  2. 找到圆形绿色二维码
  3. 扫码后选择充电时长
  4. 确认开始充电

#### 其他品牌/通用充电桩
- **扫码位置**：充电桩正面显示屏附近
- **二维码特征**：标准黑白二维码
- **操作步骤**：
  1. 查看充电桩显示屏附近
  2. 找到二维码（通常在显示屏下方或旁边）
  3. 扫码后选择充电时长
  4. 确认开始充电

## 二、常见故障处理

### 2.1 充电停不下来

**问题描述**：用户点击停止充电，但充电仍在继续

**解决方案**：

#### 特斯拉充电枪
1. **车辆端停止**：
   - 在车内点击"停止充电"
   - 或在手机App中点击"停止"
   - 等待10-20秒，充电枪会自动解锁

2. **充电桩端停止**：
   - 在充电桩显示屏点击"停止充电"
   - 等待系统响应（约10-30秒）
   - 听到"咔嗒"声表示已停止

3. **强制停止**：
   - 如上述方法无效，长按充电桩上的"急停按钮"3秒
   - 等待充电桩重启（约1分钟）
   - 再尝试拔枪

#### 比亚迪充电枪
1. **App停止**：
   - 打开比亚迪App或小程序
   - 点击"停止充电"
   - 等待充电枪解锁

2. **显示屏停止**：
   - 在充电桩显示屏点击"停止"
   - 确认停止
   - 等待解锁

3. **强制停止**：
   - 按下充电枪上的解锁按钮
   - 同时向后拔出充电枪
   - 如仍无法拔出，联系客服

#### 蔚来/小鹏/理想充电枪
1. **App停止**：
   - 在对应App中点击"停止充电"
   - 等待解锁

2. **物理停止**：
   - 长按充电桩"停止按钮"5秒
   - 或按下充电枪解锁按钮

3. **仍无法停止**：
   - 记录充电桩编号
   - 联系客服处理
   - 切勿强行拔枪

---

### 2.2 充电枪拔不出来

**问题描述**：充电结束，但充电枪卡住拔不出

**解决方案**：

#### 通用解决步骤
1. **确认已停止充电**：
   - 查看充电桩显示屏是否显示"已结束"
   - 查看车辆显示屏是否显示"充电已停止"

2. **解锁车辆**：
   - 使用车钥匙解锁车辆
   - 部分车型需要在车内按解锁键
   - 听到解锁声后再尝试拔枪

3. **正确拔枪姿势**：
   - 握住充电枪手柄
   - 按下枪上的解锁按钮（如有）
   - 向后平稳拔出，不要晃动

4. **检查是否卡锁**：
   - 如果枪头卡住，轻轻前后左右摇晃
   - 不要用力过猛，避免损坏

#### 不同品牌特殊情况

**特斯拉**：
- 车辆端必须先解锁：中控屏点击"解锁充电口"
- 充电枪会自动弹开

**比亚迪**：
- App端必须先点击"结束充电"
- 听到解锁提示音后再拔枪

**蔚来**：
- 部分站点需要手动按充电桩"解锁按钮"
- 位于充电桩侧面或显示屏下方

**仍无法拔出**：
1. 不要强行拔枪，避免损坏
2. 记录充电桩编号
3. 联系客服处理
4. 如紧急情况，可按充电桩上的"急停按钮"

---

### 2.3 充电失败/无法启动充电

**问题描述**：扫码后无法启动充电，或充电立即停止

**解决方案**：

#### 检查清单
1. **充电枪是否插好**：
   - 听到"咔嗒"声表示已插好
   - 检查是否插到底

2. **车辆是否准备就绪**：
   - 车辆电量是否已满（满电无法充电）
   - 车辆是否在充电模式（部分车型需要切换）
   - 车门是否已关闭

3. **充电桩是否正常**：
   - 显示屏是否亮起
   - 是否显示"故障"或"维护中"
   - 尝试其他充电桩

4. **支付是否正常**：
   - 检查是否绑定支付方式
   - 余额是否充足
   - 是否有未支付的订单

#### 常见错误代码

| 错误代码 | 含义 | 解决方法 |
|---------|------|---------|
| E01 | 充电枪未连接 | 重新插拔充电枪 |
| E02 | 通讯故障 | 拔枪后重新扫码 |
| E03 | 支付失败 | 检查支付方式 |
| E04 | 设备故障 | 更换其他充电桩 |
| E05 | 车辆未响应 | 检查车辆充电模式 |

---

### 2.4 充电速度慢

**问题描述**：充电速度明显低于预期

**可能原因及解决方案**：

1. **车辆端限制**：
   - 电池电量过高（>80%）会降速，属正常现象
   - 电池温度过低，预热后再充电
   - 部分车型有充电速度设置，检查是否设为"慢充模式"

2. **充电桩功率限制**：
   - 查看充电桩额定功率（60kW/120kW/180kW）
   - 部分时段会限制功率（高峰期）
   - 选择高功率充电桩

3. **同时充电车辆过多**：
   - 多车同时充电会分流功率
   - 错峰充电或选择空闲站点

## 四、其他常见问题

### 4.1 如何找到附近的充电桩？

**方法一：小程序查看**
- 打开小程序
- 点击"附近站点"
- 查看距离、空闲数量、充电速度

**方法二：地图App**
- 高德地图/百度地图搜索"充电站"
- 查看实时空闲状态

### 4.2 充电需要预约吗？

- 大部分站点无需预约，即到即充
- 热门站点建议提前查看空闲状态
- 部分站点支持预约功能（小程序内操作）

### 4.3 可以用其他品牌的充电桩吗？

- 可以，本平台支持大部分主流车型
- 扫码后选择您的车型即可
- 部分品牌专用桩可能有兼容性问题

### 4.4 充电时可以在车里吗？

- **可以**，但建议：
  - 保持通风
  - 不要使用大功率电器（空调等）
  - 注意安全，不要吸烟或使用明火

### 4.5 充电时长建议

- **快充桩**：30-60分钟（充至80%）
- **慢充桩**：4-8小时（充满）
- 建议充至80%即可，保护电池

## 六、安全提示

### 6.1 充电安全

1. **充电前检查**：
   - 充电枪是否完好
   - 充电口是否清洁干燥
   - 周围是否有易燃物

2. **充电中注意**：
   - 不要强行拔枪
   - 不要移动车辆
   - 发现异常立即停止充电

3. **充电后确认**：
   - 充电枪是否归位
   - 充电口盖是否盖好
   - 确认充电已结束

### 6.2 紧急情况处理

**发现冒烟/异味/火花**：
1. 立即停止充电（按急停按钮）
2. 远离充电桩（至少10米）
3. 拨打客服电话或报警
4. 不要自行处理

**触电/火灾**：
1. 立即切断电源（急停按钮）
2. 拨打119/120
3. 不要用水灭火
