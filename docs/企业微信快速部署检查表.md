## 🚀 快速部署步骤（30分钟）

### 第一步：创建应用（5分钟）

| 序号 | 操作步骤 | 完成标记 | 备注 |
|------|---------|----------|------|
| 1.1 | 访问 https://work.weixin.qq.com/ | ☐ | 管理员扫码登录 |
| 1.2 | 进入「应用管理」→「自建」→「创建应用」 | ☐ | |
| 1.3 | 填写应用名称：`充电桩智能客服` | ☐ | |
| 1.4 | 上传应用Logo | ☐ | 240×240px |
| 1.5 | 填写应用介绍 | ☐ | 简短描述功能 |
| 1.6 | 选择测试部门/成员 | ☐ | 先小范围测试 |
| 1.7 | 点击「创建应用」 | ☐ | |

---

### 第二步：获取凭证（5分钟）

⚠️ **重要**：以下信息请立即复制保存！

| 序号 | 操作步骤 | 获取信息 | 完成标记 |
|------|---------|----------|----------|
| 2.1 | 在应用详情页顶部获取 | **AgentId** | ☐ |
| 2.2 | 点击「获取」Secret | **Secret** | ☐ | 仅显示一次！ |
| 2.3 | 进入「我的企业」→「企业信息」 | **CorpId** | ☐ |

**将以下信息发送给技术方**：
```
企业微信配置：
├─ CorpId: [你的企业ID]
├─ AgentId: [你的应用ID]
└─ Secret: [你的应用密钥]
```

---

### 第三步：配置接收消息（10分钟）

**前置条件**：技术方已提供腾讯云函数地址

| 序号 | 操作步骤 | 完成标记 | 备注 |
|------|---------|----------|------|
| 3.1 | 在应用详情页找到「接收消息」 | ☐ | |
| 3.2 | 点击「设置接收消息」 | ☐ | |
| 3.3 | 填写URL：[腾讯云函数地址] | ☐ | 技术方提供 |
| 3.4 | 填写Token：[自定义字符串] | ☐ | 任意字符串 |
| 3.5 | 点击「随机生成」EncodingAESKey | ☐ | |
| 3.6 | 点击「保存」并等待验证 | ☐ | 显示「验证成功」即可 |

---
