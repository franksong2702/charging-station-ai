# 企业微信部署配置信息交换模板

> **用途**：客户方与技术方之间交换部署所需配置信息  
> **版本**：v1.0  
> **最后更新**：2026-04-03

---

## 📋 信息传递说明

本文档用于客户方和技术方之间安全地交换企业微信部署所需的配置信息。

### 信息安全提醒

⚠️ **重要**：
- Secret 等敏感信息请通过加密渠道传递
- 不要在公开的 Git 仓库中提交此类信息
- 建议使用企业微信、邮件等内部渠道传递

---

## 📤 客户方 → 技术方

### 企业微信配置信息

请客户方填写以下信息并提供给技术方：

```
【企业微信配置信息】
════════════════════════════════════════
企业ID (CorpId): 
────────────────────────────────────────
应用ID (AgentId): 
────────────────────────────────────────
应用密钥 (Secret): 
────────────────────────────────────────
【注意】Secret 仅显示一次，请立即复制！
════════════════════════════════════════
```

### 应用基本信息（可选）

```
【应用基本信息】
════════════════════════════════════════
应用名称: 充电桩智能客服
────────────────────────────────────────
应用Logo: (请提供Logo文件)
────────────────────────────────────────
应用介绍: 为企业员工提供充电桩使用咨询、故障处理等智能客服服务
────────────────────────────────────────
可见范围: (请描述目标用户范围)
════════════════════════════════════════
```

## 📥 技术方 → 客户方

### 腾讯云函数配置信息

请技术方填写以下信息并提供给客户方：

```
【腾讯云函数配置】
════════════════════════════════════════
接收消息 URL: 
────────────────────────────────────────
(示例: https://ap-guangzhou.apigw.tencentcs.com/release/charging-station-customer-service)
════════════════════════════════════════
```

### Coze 平台配置信息（可选）

```
【Coze 平台配置】
════════════════════════════════════════
工作流名称: 充电桩智能客服
────────────────────────────────────────
工作流 ID: (如需要)
────────────────────────────────────────
API 访问地址: (如需要)
════════════════════════════════════════
```

### 验证 Token（可选）

```
【验证配置】
════════════════════════════════════════
Token (用于企业微信验证): 
────────────────────────────────────────
(可自定义，如: charging_station_2024)
════════════════════════════════════════
```
