### TC-01 知识库问答-使用指导

**测试目的**：验证知识库问答功能正常工作

**测试步骤**：

```
用户输入：特斯拉怎么充电？
```

**预期结果**：

## 附录

### A. 意图识别关键词速查

```
使用指导（usage_guidance）：
- 默认意图，无明显故障关键词

故障处理（fault_handling）：
- 充不进去、充不上、充不进、充不了
- 充电失败、无法充电、无法启动
- 坏了、故障、异常

兜底流程（fallback/complaint）：
- 投诉、垃圾、骗子
- 转人工、人工客服
- 强烈不满表达

不满意（dissatisfied）：
- 没用、不行、没帮助
- 没解决问题

满意（satisfied）：
- 谢谢、感谢、满意
- 很好、不错

评价反馈（feedback_good/feedback_bad）：
- 1、很好、有帮助 → feedback_good
- 2、没有帮助、没用 → feedback_bad

取消兜底（cancel_fallback/exit_fallback）：
- 算了、再见、不聊了
- 不玩了、不用了、不需要了
```

### B. 兜底流程状态图

```
fallback_phase = "" (初始状态)
        ↓
        用户表达强烈不满 / 直接提供手机号+车牌号
        ↓
fallback_phase = "collect_info" (收集信息)
        ↓
        收集到手机号 + 车牌号
        ↓
fallback_phase = "confirm" (等待确认)
        ↓
        用户确认
        ↓
fallback_phase = "done" (已完成)
        ↓
        下次对话重新开始（不保存 done 状态）
```
